# Contributing to NutriScan-AI

Thank you for your interest in contributing to NutriScan-AI! We welcome contributions from developers of all skill levels to help make Indian food recognition even better.

## 🚀 How Can I Contribute?

### 1. Reporting Bugs
- Use the [GitHub Issues](https://github.com/kalyan-1845/NutriScan-AI/issues) tab to report bugs.
- Provide a clear description and steps to reproduce.

### 2. Suggesting Features
- We love new ideas! Open an issue with the "feature request" tag.

### 3. Code Contributions
1. **Fork** the repository.
2. **Clone** your fork: `git clone https://github.com/your-username/NutriScan-AI.git`
3. **Create a branch**: `git checkout -b feature/amazing-feature`
4. **Commit** your changes: `git commit -m "feat: Add amazing feature"`
5. **Push** to the branch: `git push origin feature/amazing-feature`
6. **Open a Pull Request**.

## 🛠️ Development Setup
- Follow the instructions in [README.md](README.md) to set up the Python backend and React frontend.
- Ensure your code follows the established design patterns (Glassmorphism UI, Flask REST patterns).

## 📜 Code of Conduct
Please be respectful and professional in all interactions within the NutriScan-AI community.
